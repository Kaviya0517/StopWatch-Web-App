# Simple Stopwatch ⏱

A simple stopwatch web application built using HTML, CSS, and JavaScript.

## Features
- Start the stopwatch
- Stop the stopwatch
- Reset the stopwatch
- Displays time in Hours : Minutes : Seconds
- Clean card UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Preview
A simple stopwatch interface with start, stop and reset buttons.
